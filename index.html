Playz Rails Music Player

Playz is an music player that allows one to play music from your home-served music collection. Your music remains on your computer, but is indexed on the playz server. Features are:
  • Backend is entirely Ruby on Rails
  • Very fast search using SOLR technology.
  • In addition to song, album and artist, also searches by song lyrics.
  • Plays using HTML5 audio if available, otherwise falls back to flash.
  • Standard play controls
  • Easy interface for playlist manipulation

Rio Library for Ruby

Rio is a facade for most of the standard ruby classes that deal with I/O; providing a simple, intuitive, succinct interface to the functionality provided by IO, File, Dir, Pathname, FileUtils, Tempfile, StringIO, OpenURI and others. Rio also provides an application level interface which allows many common I/O idioms to be expressed succinctly.
